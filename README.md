# Install

Get binary from [releases](https://github.com/thewh1teagle/serust/releases)

Install with winget
```console
winget install Thewh1teagle.Serust
```
Install with `cargo`
```console
cargo install serust
```

# Usage
You can get info by
```console
serust -h
```

# Build
```console
git clone https://github.com/thewh1teagle/serust
cd serust
cargo build --release
```
